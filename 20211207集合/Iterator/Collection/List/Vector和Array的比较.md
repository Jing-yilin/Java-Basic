# Vector和ArrayList的比较

|           | 底层结构 | 版本 | 线程安全（同步）效率 | 扩容倍数                                                     |
| --------- | -------- | ---- | -------------------- | ------------------------------------------------------------ |
| ArrayList | 可变数组 |      | 不安全、效率高       | 如果是有参构造器，则按照1.5倍扩容；如果是无参构造器，第一次容量为10，第二次开始按照1.5倍扩容 |
| Vector    | 数组     |      | 安全、效率低         | 如果是无参，默认10，满后，按照2被扩容；如果是指定容量大小，则每次按照2倍扩容 |
